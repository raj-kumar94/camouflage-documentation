# Hide specific variants based on countries

Pick specific variants and hide them only from shoppers in certain countries — useful when:

* A supplier ships some colours / sizes only to certain regions.
* You're restricted from selling certain variants in a country for legal or licensing reasons.
* You want a region-exclusive launch ("US-only colourway" while the rest of the world sees the standard range).

You can hide the variant from a list of countries, OR from **all** countries except a list you provide.

{% hint style="info" %}
The shopper's country is read from Shopify's localisation. Make sure your store has the **Markets** / **Localisation** feature configured so Shopify knows which country to assign to each visitor.
{% endhint %}

### Before you select and hide any variants

1. Make sure the theme setup is done correctly. Refer to [Step 1](../camouflage-setup-guide/basic-configuration.md)
2. Go to the Camouflage app -> `Hide specific variants` page -> `Select variants individually` tab
3. If you want to hide only the specifically selected variants, tick the checkbox `Hide only specifically selected variants` at the top of the page.



### Let's hide specific variants based on countries

1. Go to the Camouflage app -> `Hide specific variants` page -> `Select variants individually` tab
2. Search for a product
3. Click on the **"Select variants"** button
4.  Select variants to hide and click "Hide from countries"<br>

    <figure><img src="../.gitbook/assets/image (39).png" alt=""><figcaption></figcaption></figure>
5.  Select the countries where you want the selected variants hidden and click the "Hide from countries" button.<br>

    <figure><img src="../.gitbook/assets/image (40).png" alt=""><figcaption></figcaption></figure>
6.  If you want to Hide from all countries except a few, then leave the `Hide selected variants from these countries` field empty and add countries in the `But not from these countries` field instead<br>

    <figure><img src="../.gitbook/assets/image (41).png" alt=""><figcaption></figcaption></figure>

###

### Make the hidden variants visible again

1. Go to the Camouflage app -> `Hide specific variants` page -> `Select variants individually` tab
2. Search for a product
3. Click on **"Select variants"** button
4. Select the variants to show again. Hidden variants will display a "Hidden in" badge next to them
5. Click on 3 dots at the top right corner to expand more actions
6. Click **"Undo hide from countries"**

***

**Need help?** If the variant isn't hidden for shoppers in the expected country, see [Troubleshooting](../troubleshooting.md) — check Shopify's localisation setup first. Still stuck? Open the in-app chat from your Camouflage dashboard.
