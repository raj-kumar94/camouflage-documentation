# Hide specific variants regardless of inventory quantity

You can cherry-pick any variants to hide. If you have hundreds of products where a common variant or product option needs to be hidden, that's possible too. You can also select countries or customer tags for which variants should be hidden.



### Before you select and hide any variants

1. Make sure the theme setup is done correctly. Refer to [Step 1](../../camouflage-setup-guide/basic-configuration.md)
2. Go to the Camouflage app -> `Hide specific variants` page -> `Select variants individually` tab
3. If you want to hide only the specifically selected variants, tick the checkbox `Hide only spefically selected variants` at the top of the page.



### Let's hide a specific variant

1. Go to the Camouflage app -> `Hide specific variants` page -> `Select variants individually` tab
2. Search for a product
3.  Click on **"View variants"** button<br>

    <figure><img src="../../.gitbook/assets/image (4).png" alt=""><figcaption></figcaption></figure>
4.  Select the variants to hide and then click on "Always hide"<br>

    <figure><img src="../../.gitbook/assets/image (6).png" alt=""><figcaption></figcaption></figure>



### Make the hidden variants visible again

1. Go to the Camouflage app -> `Hide specific variants` page -> `Select variants individually` tab
2. Search for a product
3. Click on **"View variants"** button
4. Select the variants to show again. Hidden variants will display a "Always hide" badge next to them
5. Click on 3 dots at the top right corner to expand more actions
6. Click **"Undo always hide"**
