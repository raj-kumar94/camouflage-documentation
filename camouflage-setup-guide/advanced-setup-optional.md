# 3️ ADVANCED SETUP (optional)

{% hint style="info" %}
Go to the Setup page -> Advance Setup tab to view the settings
{% endhint %}

<figure><img src="../.gitbook/assets/image (13).png" alt=""><figcaption><p>Camouflage - Advance setup</p></figcaption></figure>

## Setting Exceptions

Add tags to set exceptions. Camouflage will not hide/disable/strike-through etc on the tagged products.

Eg: If you add the tag `camouflage-exclude` , Camouflage will not do anything on the products that are tagged with `camouflage-exclude`

<figure><img src="../.gitbook/assets/image (14).png" alt=""><figcaption></figcaption></figure>

## Setting Conditions

Add tags to set conditions. Camouflage will hide/disable/strike-through etc only on the tagged products.

Eg: If you add the tag `new-arrivals` , Camouflage will work only on the products that are tagged with `new-arrivals`

<figure><img src="../.gitbook/assets/image (15).png" alt=""><figcaption></figcaption></figure>
