# camouflage-documentation

Customer-facing GitBook for the Camouflage Shopify app, git-synced. **Pushing `main` publishes live to customers — never push without explicit approval.**

**Read first, before editing any page:** `.claude/skills/camouflage-docs/SKILL.md` — publishing model, section map, GitBook syntax conventions, and the audience rules (merchant pages must never mention internal implementation like `custom_js`; developer pages document only the public `window.hide_oos_*` surface). Every new page must be added to `SUMMARY.md`.

The app itself lives at `../camouflage-app` (skills at `../camouflage-app/.claude/skills/`) — verify technical claims there before documenting them.
